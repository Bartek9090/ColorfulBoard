# ColorfullBoard

## Description
The application works by changing the colors of the squares when the cursor hovers over them.
App has been created to practice and learn new js tricks.

## About Project 
Projects give a possibilities to choose:
- 4 color options
- 3 speed options
- level of saturation
  
## Technologies:
* HTML 5
* CSS
* JS
